---
title: Programs and hosts
description: Organize show metadata and presenter names for the schedule.
---

**Programs** and **Hosts** are reusable libraries that make scheduling faster and give your calendar consistent labels.

## Programs

A program is a show entry with:

- **Title**
- **Description**
- **Host** (linked to a host name)

### Managing programs

1. Open **Programs** in the sidebar.
2. Click **New program** to create one.
3. Fill in title, description, and host.
4. Save.

You can edit or delete programs from the same view.

### Using programs on the schedule

When creating or editing a schedule slot, search for and select a program. The slot inherits the program's title, description, and host.

You can still override the title or description on individual slots after selecting a program.

## Hosts

A host is a named presenter with a calendar color. Hosts appear on schedule slots and in the player bar.

### Managing hosts

1. Open **Hosts** in the sidebar.
2. Click **New host**.
3. Enter a name and pick a color.
4. Save.

Edit or delete hosts from the list. Deleting a host does not remove historical slot references, but the host will no longer appear in pickers.

### Using hosts on the schedule

When editing a slot, assign one or more hosts from the multi-select field. Host colors appear on the calendar for quick visual identification.

You can also create a new host inline while editing a program or schedule slot.

## Relationship between programs and hosts

- Each program has one primary host name.
- Schedule slots can list multiple hosts.
- Host names are shared across programs, slots, and the hosts library.

## Tips

- Create hosts before programs so program creation can link to them.
- Use programs for recurring shows with stable titles and descriptions.
- Use host colors to distinguish presenters at a glance on busy schedule days.
